256color-tmux-tty
=================

Files I use to run tmux with 256 colors in tty1 (No X session). Only uses ~60MB of RAM!

![Screenshot](screenshot.png)

Requirements
============

fbterm (Using version 1.7 [recommended])

tmux (Using 1.9a [recommended])
